Product Requirements Document

Purpose:
To create an engaging, educational web-based animal farm game where users can interact with animals and plants, complete tasks.

Scope:
Interactive farm with animals (Cat, Dog, Fish) and plants (Apple Tree, Banana Tree)
Inventory system for collected items
Shop to buy items using resources
Mini-games for rewards (Word Puzzle, Reaction Game, Memory Game)
Leveling system with experience points
Daily tasks and rewards

user:
Kindergarten and primary school students

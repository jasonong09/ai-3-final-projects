# Jason's AI Projects

Welcome to the repository for my AI-assisted web projects. This collection features interactive games and educational tools built with web technologies.

## Projects

### 1. [Animal Farm](./animal-farm/)
An interactive game where you manage a farm, feed animals, and complete daily missions.
- **Features**:
  - Feed and grow animals (Cat & Dog)
  - Inventory system
  - Save/Load functionality
  - Daily missions and rewards
  - Shop system

### 2. [Smart Farm Assistant](./plant-game/)
An educational tool and mini-game focused on agriculture and gardening.
- **Features**:
  - Crop gallery and information
  - Garden space calculator
  - Chat interface for farming advice
  - "Growing Tips" mini-game

## How to Run
Simply open the `index.html` file in the root directory to access the project portal, or navigate to the specific project folders to run them individually.
